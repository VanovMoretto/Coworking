# Coworking reservations

### This website is an internal room booking environment. The site offers four different spaces for making reservations. Also, you can add the participants for the reservation, view your completed bookings and reservation history. Additionally, you can make a reservation for events in the Arena space. You can also request additional orders, room preparation, etc. All of this is done in such a way that your user is online, therefore, providing better control of the booking made by you.

## Installation

To install the project locally, follow these steps:

1 - Clone the repository:
git clone https://github.com/VanovMoretto/Coworking.git

2 - Install dependencies:
cd coworking
npm install

3 - Start the local server:
npm start
The website will be available at http://localhost:3000.

## Usage
To begin, you must first create your account, after which you can choose among the spaces you want to book. There are 3 locations that you can reserve by yourself, simply by selecting the time and date and making the reservation. For the Arena, you need to contact the events coordinator to request the reservation by filling out a form. You can access your bookings via User>My Reservations. Also, you can change your account details via User>My Account.

# Reservas de Coworking (Versão em Português)

### Este site é um ambiente interno de reservas de sala. O site oferece quatro espaços diferentes para fazer reservas. Além disso, você pode adicionar os participantes da reserva, visualizar suas reservas concluídas e o histórico de reservas. Adicionalmente, você pode fazer uma reserva para eventos no espaço da Arena. Você também pode solicitar pedidos adicionais, preparação de sala, etc. Tudo isso é feito de forma que seu usuário esteja online, portanto, proporcionando um melhor controle da reserva feita por você.

## Instalação
Para instalar o projeto localmente, siga estas etapas:

1 - Clone o repositório:
git clone https://github.com/VanovMoretto/Coworking.git

2 - Instale as dependências:
cd coworking
npm install

3 - Inicie o servidor local:
npm start

O site estará disponível em http://localhost:3000.

## Uso
Para iniciar, você deve primeiro criar sua conta, após isso, é possível escolher entre os ambientes que você quer reservar. Há 3 locais que você pode reservar por você mesmo, apenas selecionando a hora e a data e efetuando a reserva. Para a Arena, é necessário entrar em contato com a coordenadora de eventos para solicitar a reserva por meio de preenchimento de um formulário. Você consegue acessar suas reservas através do usuário>minhas reservas. E também, consegue alterar seus dados da conta através do usuário>minha conta.
